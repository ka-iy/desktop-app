; IVPN Windows Release Manifest
; Used by build.bat to stage files into the installer temp directory.
;
; Format: DEST[=SOURCE]
;   DEST   - destination path in the installer staging dir (flat, no arch subdir)
;   SOURCE - source path relative to the project root (optional)
;            if omitted, SOURCE is assumed to be the same path as DEST
;            relative to the project root
;            {ARCH} is replaced with the target arch (x86_64 or arm64)
;
; Optional line prefix [x86_64] or [arm64] limits the entry to that arch only.
; Lines starting with ; are comments and are ignored.
